Connection Matrix:

For Button to Arduino Pins:
Red Button -> Pin 13
Blue Button -> Pin 12
Yellow Button -> Pin 11
Green Button -> Pin 10
White Button -> Pin 9

For LED to Arduino Pins:

Red LED -> Pin 6
Blue LED -> Pin 5
Yellow LED -> Pin 4
Green LED -> Pin 3
White LED -> Pin 2

+ve buzzer terminal to arduino -> Pin 1

Correct (Green) LED to arduino -> Pin 8
Wrong (Red) LED to arduino -> Pin 7
